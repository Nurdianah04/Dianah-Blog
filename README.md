# Personal Blog Page Portfolio 📝

## 1. Project Introduction

This project is a Personal Blog Page developed for the individual assignment of the **CSD34203 Special Topics in Software Development** course at Universiti Sultan Zainal Abidin (UniSZA).

The website serves as a digital portfolio to showcase personal information, technical skills, and blog articles related to technology, web development, and learning experiences in the IT field.

---

## 2. Main Features

* **Responsive Design**
  The website layout is optimized for multiple screen sizes including smartphones, tablets, and desktop devices.

* **Dark Mode Toggle**
  Users can switch between light and dark themes using JavaScript. The selected theme is saved using `localStorage`.

* **Multi-Page Navigation**
  The website contains several main pages:

  * Home
  * About Me
  * Blog
  * Contact

* **Interactive Blog Section**
  Displays sample blog posts related to technology and software development.

* **Contact Form**
  Users can send feedback or messages through the Contact page.

---

## 3. Technologies Used

* **HTML5** – Used to structure the website content
* **CSS3** – Used for styling and responsive layout design
* **JavaScript (ES6)** – Used for interactive features such as Dark Mode and DOM manipulation
