# Personal Blog Page Portfolio 📝

## 1. Pengenalan Projek
Projek ini merupakan sebuah laman blog peribadi (Personal Blog Page) yang dibangunkan khusus untuk tugasan individu kursus **CSD 34203 Special Topics in Software Development** di Universiti Sultan Zainal Abidin (UniSZA). Projek ini berfungsi sebagai portfolio digital untuk memaparkan profil peribadi, kemahiran teknikal, serta artikel penulisan berkaitan teknologi dan pembangunan perisian.

---

## 2. Ciri-Ciri Utama (Features)
*   **Reka Bentuk Responsif (Mobile-First Layout):** Antara muka yang kemas dan dioptimumkan untuk pelbagai saiz skrin, termasuk telefon pintar, tablet, dan komputer desktop.
*   **Sistem Mod Gelap (Dark Mode Toggle):** Interaksi dinamik menggunakan JavaScript yang membolehkan pengguna menukar tema warna mengikut keselesaan mata, lengkap dengan storan tempatan (`localStorage`) untuk menyimpan tetapan pilihan pengguna.
*   **Navigasi Berasingan Multi-Halaman:** Pengurusan struktur halaman yang bersih merangkumi halaman Utama, Tentang Saya, Blog, dan Hubungi Saya.
*   **Borang Hubungi Interaktif:** Borang maklum balas yang mesra pengguna pada halaman Hubungi Saya.

---

## 3. Teknologi yang Digunakan (Technologies Used)
*   **HTML5:** Untuk penstrukturan semantik kandungan web.
*   **CSS3:** Untuk penggayaan, reka bentuk grid/flexbox responsif, dan pengurusan pemboleh ubah warna (CSS Variables).
*   **JavaScript (ES6):** Untuk logik manipulasi DOM (pertukaran mod tema) dan penyimpanan data sesi.
